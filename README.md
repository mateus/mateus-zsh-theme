### Install

> Requires https://github.com/robbyrussell/oh-my-zsh

Copy `mateus.zsh-theme` to `~/.oh-my-zsh/themes`

```
 Â $ ~/.oh-my-zsh/themes/mateus.zsh-theme
```

Change the theme in `~/.zshrc`

```
  ZSH_THEME="mateus"
  . . .
```

Load `~/.zshrc`

```
  $ source ~/.zshrc
```
